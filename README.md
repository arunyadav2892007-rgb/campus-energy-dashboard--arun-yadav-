# campus-energy-dashboard--arun-yadav-
Campus Energy-Use Dashboard

A Python-based capstone project that reads multiple building energy CSV files, validates the data, performs time-series analysis, generates visual dashboards, and produces summary reports for campus energy monitoring.

Features

Load multiple CSV files automatically

Validate corrupted/missing data

Daily & weekly energy aggregation

Building-wise summary (mean, min, max, total)

Object-Oriented modeling (Building, MeterReading, Manager)

Dashboard with:

Daily trend line

Weekly building comparison (bar chart)

Peak load scatter plot

Export cleaned dataset & summary CSV

Auto-generate executive summary report

How to Run
python energy_dashboard.py
